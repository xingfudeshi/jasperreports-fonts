# jasperreports-fonts
A Chinese fonts extension for jasperreports.
Font contained:  
Source Han Sans K Regular  
Source Han Sans K Medium 


Quick start:

``` java
<dependency>
  <groupId>com.github.xingfudeshi</groupId>
  <artifactId>jasperreports-fonts</artifactId>
  <version>1.2.0</version>
</dependency>
```
